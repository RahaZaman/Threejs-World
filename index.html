Camera Controls

  • Left Mouse Button + Drag: Rotate camera around the scene
  • Right Mouse Button + Drag: Pan camera
  • Mouse Wheel: Zoom in/out
  • Shift + Drag: Move faster
  • Ctrl + Drag: Move slower

Interactive Objects

  • Left Mouse Button + Click on an object: Change object color randomly (Wow feature)
